Compiled ECS Feature Runtime

ECS 保存权威状态，初始化期编译 Entity、Projection 与 Effect 能力，业务 Feature 只组装真实拓扑和池。

运行流

GameScene.init
  -> EntityRuntime / ProjectionRuntime / EffectRuntime
  -> bootstrap singleton entities
  -> Feature setup creates topology and pools

frame
  -> state systems
  -> network
  -> feature systems
  -> projection mark / sync
  -> effect flush

三个运行时

EntityRuntime
基数、组件组合、初始化、固定槽位。
ProjectionRuntime
预编译 query、snapshot、差量 row 和 node registry。
EffectRuntime
按 definition identity 派发瞬时事实。

业务新增路径

  1. 定义 component 和 EntityType。
  2. 编写纯 system。
  3. 持久表现定义 Projection，瞬时表现定义 Effect。
  4. Feature 声明 entities/systems/projections。
  5. setup 显式创建 topology、池和节点。

设计约束

  • 一般一个 world，退出时整批清理，进入时重建。
  • 运行期通常不 removeEntity，优先状态复用和池。
  • 业务不维护 dirty bit、registry key、full sync 或全局字符串事件。
  • Feature 是 assembly，不是规则容器。
